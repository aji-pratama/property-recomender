# Propertify is Recomendation System to find the best property based on keyword using Cosine Similiarity TF-IDF Method

#### Prerequisite
- Python >3.6
- Java/JDK 


### Install Package
`$ pip install -r requirements.txt`


### Run Applaction
1. Run mongoDB
2. Run Crawler: 
    - ` $ scrapy crawl dotproperty`
    - ` $ scrapy crawl rumahcom`
    - ` $ scrapy crawl lamudi `

3. Run Appliaction
    - ` $ python run.py`
